A short rest removes one point of exhaustion, a long rest removes three points of exhaustion.

| Level | Effect                      |
| ----- | --------------------------- |
| 1     | -1 to d20                   |
| 2     | -2 to d20, -5 Movement      |
| 3     | -3 to d20,                  |
| 4     | -4 to d20, -10 Movement     |
| 5     | -5 to d20,                  |
| 6     | -6 to d20, Half move speed  |
| 7     | -7 to d20,                  |
| 8     | -8 to d20, Max HP is halved |
| 9     | -9 to d20,                  |
| 10    | -10 to d20, 0 Move Speed    |
| 11    | Death                       |
